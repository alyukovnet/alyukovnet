Сборка и запуск

Зависимости

  1. g++ compiler
  2. CMake
  3. Catch2 (включён в проект)

Сборка

mkdir build
cd build
cmake .. && cmake --build

Тестирование

ctest

Запуск примеров

./examples/cli
